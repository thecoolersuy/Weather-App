## AQI Monitor App / Weather Forcast

A minimal AQI monitor and weather forcast app made purely on HTML, CSS and JavaScript to monitor and observe the AQIs of different locations using an AQI fetch API and a Weather Detection as well made on Javascript DOM utilizing Javascript ES6 features and proper CSS format.
